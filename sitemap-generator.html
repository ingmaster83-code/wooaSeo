Sitemap XML 생성기 — URL 목록으로 사이트맵 자동 생성

URL 목록을 붙여넣으면 sitemap.xml을 즉시 생성합니다. lastmod, changefreq, priority 설정 후 바로 다운로드.

URL 목록 입력
생성된 sitemap.xml

💡 Sitemap 작성 팁

  • 홈페이지(/)는 priority 1.0, 변경이 잦은 페이지는 daily, 정적 페이지는 monthly를 권장합니다.
  • 생성된 sitemap.xml을 서버 루트(/)에 업로드하고 robots.txt에 Sitemap 경로를 추가하세요.
  • Google Search Console에서 sitemap을 제출하면 색인 속도가 빨라집니다.
  • 50,000개 이상의 URL은 여러 개의 sitemap으로 분리하고 sitemap index를 사용하세요.

자주 묻는 질문

sitemap.xml은 어디에 올려야 하나요?
웹사이트 루트 디렉터리에 업로드하세요 (예: https://example.com/sitemap.xml). 그 후 robots.txt에 "Sitemap: https://example.com/sitemap.xml" 한 줄을 추가하면 크롤러가 자동으로 인식합니다.
priority 값은 어떻게 설정하나요?
0.0~1.0 범위로 설정하며 홈페이지는 1.0, 주요 페이지는 0.8~0.9, 일반 페이지는 0.5~0.7, 중요도 낮은 페이지는 0.3 이하를 권장합니다. Google은 이 값을 참고만 합니다.
Google Search Console에 sitemap을 어떻게 제출하나요?
Search Console → 해당 속성 선택 → 왼쪽 메뉴 "Sitemaps" → sitemap URL 입력 후 제출 버튼을 클릭하면 됩니다.
