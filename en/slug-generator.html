💡 URL Slug Tips

  • Use hyphens (-) as separators. Google treats hyphens as word separators, but underscores (_) are not recognized the same way.
  • Keep slugs short and descriptive. 3–5 keywords is ideal.
  • Remove stop words (a, an, the, of, in) to keep slugs clean.
  • Avoid dates in slugs (/2024/01/my-post) — they're hard to change later.
  • Once published, changing a slug breaks existing links. Always set up a 301 redirect if you must change it.

Frequently Asked Questions

Do URL slugs affect SEO?
Yes. Keywords in the URL are a weak ranking signal for Google. More importantly, descriptive URLs improve click-through rates (CTR) because users can understand what the page is about from the URL alone.
What happens if I change an existing slug?
The old URL will return a 404 error. Always set up a 301 redirect from the old URL to the new one to preserve SEO value and avoid broken links.
Should I use hyphens or underscores?
Use hyphens. Google's John Mueller has confirmed that hyphens are treated as word separators, while underscores join words together. So "image-compressor" is better than "image_compressor" for SEO.
